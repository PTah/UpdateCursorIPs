# UpdateCursorIPs

Небольшая утилита для обновления текстового файла со списком IP-адресов.

Основной сценарий:
- прочитать текущий список адресов из файла;
- добавить новые IP;
- удалить указанные IP;
- записать результат обратно без дублей и в отсортированном виде.

Подробный пример запуска приведён в `USAGE.md`.
